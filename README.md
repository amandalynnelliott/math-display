# Getting Started

## Setup

Download and install [Git](https://git-scm.com/) and [Node.js](https://nodejs.org/).

```sh
git clone https://github.com/amandalynnelliott/math-display.git
cd math-display
npm install
```

## Build

```sh
npm run build
```

## Watch

```sh
npm run watch
```

## Run Tests

```sh
npm test
```
